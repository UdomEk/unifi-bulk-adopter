# UniFi Bulk AP Adopter

A Telegram Mini App for adopting UniFi Access Points in bulk.

## Features

- 🔐 Secure PIN-based login
- 👥 User management (Super Admin + Admins)
- 📡 Bulk AP adoption via SSH
- 🔄 Retry failed adoptions
- 📊 Progress tracking

## Deployment

This app is deployed on Vercel and uses Supabase for user management.
